## 🔬 Featured Projects

### Project 1 — Vulnerability Scans (Nessus / Nmap / OpenVAS)
Conducted layered vulnerability assessments across target systems. Used Nessus and OpenVAS for in-depth scanning and Nmap for network enumeration and OS detection. Documented findings in a penetration testing report with mitigation recommendations.

**Tools:** Nessus · Nmap · OpenVAS

### Project 2 — Identifying & Analyzing Malicious Activity
Performed remote Linux system administration via SSH, monitored processes using `top` and `htop` (EPEL), implemented Active Directory account auditing, applied Group Policy updates, and analyzed Security Event Logs in Event Viewer to detect unauthorized privilege escalation.

**Tools:** PowerShell · SSH · Active Directory · Event Viewer · htop · gpupdate
